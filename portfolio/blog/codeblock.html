Code Snippets in HTML

Writing about code it fun! But what's they best way to display your code snippet on an HTML page? This post will cover the appropriate tags to use, and how to add Prism to HTML pages & WordPress themes to give your code snippets beautiful syntax highlighting.

The simplest way to display code snippets in HTML is by using the "pre" and "code" tags like this:

CODE

<pre><code>&lt;p&gt;Hello World!&lt;\p&gt;</code></pre>

RESULT

<p>Hello World!</p>

Notice that in the "pre code" block character encoding must be taken into account to display the following:

  • Ampersand: &amp;
  • Less than: &lt;
  • Greater than: &gt;

A "pre code" block preserves the indentation and line spacing, which makes the code blocks easier to read and is useful for languages such as Python where indentation is critically important.

The "pre" and "code" tags should be aligned normally with other tags on the page. Start typing the code on the same line with no extra whitespace. New lines should be aligned with the left edge of the page, not keep inline with the other html code.

CODE

<main>
    <section>
        <article>

@lru_cache()
def fibonacci(n):
    if type(n) != int:
        raise TypeError("n must be a positive int")
    if n < 1: 
        raise TypeError("n must be a positive int") 
    if n==1: 
        return 1 
    elif n==2: 
        return 1
    elif n > 2:

RESULT

from functools import lru_cache
@lru_cache()
def fibonacci(n):
    if type(n) != int:
        raise TypeError("n must be a positive int")
    if n < 1:
        raise TypeError("n must be a positive int")
    if n == 1:
        return 1
    elif n == 2:
        return 1
    elif n > 2:
        return fibonacci(n-1) + fibonacci(n-2)

Style with Prism

Prism is a simple, beautiful, lightweight syntax highlighter. It's compatible with hundreds of languages and offers eight color themes. You can check out the themes here. This is what the Python code above looks like using Prism with the "Tomrrow Night" theme: 

from functools import lru_cache
@lru_cache()
def fibonacci(n):
    if type(n) != int:
        raise TypeError("n must be a positive int")
    if n < 1:
        raise TypeError("n must be a positive int")
    if n == 1:
        return 1
    elif n == 2:
        return 1
    elif n > 2:
        return fibonacci(n-1) + fibonacci(n-2)

Much nicer!

2 options to include Prism in an html project:

1. Link the style sheets with a CDN

You can get the necessary links from cdnjs. Copy the links for the "prism-core.min.js" file as well as the css file for your chosen theme and add them to the head of your html page. This option is quick and easy but may slow down your page. It also won't work well for a CMS, such as WordPress.

2. Download the files and add them to your project folder

Prism is simple to download and allows you to select only the languages and features you need to keep it as lightweight as possible. Go here to select your theme, languages, addtional featueres and download the js and css files. For my examples I included the "Copy to Clipboard Button" which is in "Plugins", the last section. Other helpful plugins include "Line Numbers", to print the line numbers and "Unescaped Markup", to avoid having to replace "<" with "&lt;".

Using Prism

Once you have the files downloaded, link them by path in the head of your html page. If you put the files in an assests folder it might look something like this:

<link rel="stylesheet" href="./assets/prism.css"/>
<script src="./assets/prism.js"></script>

Once you have Prism linked in your project all it takes to display fancy syntax highlighting is adding a class to the code tag:

<pre><code class="language-python">...
...
</code></pre>

Somehting to keep in mind is that the class for HTML is "language-markup", which will also work for XML, SVG, MAthML, SSML, Atom, and RSS.

Prism and WordPress

WordPress has many plugin options for styling code snippets, so why choose to add Prism instead? If you are concerned about speed and want to use as few plugins as possible, or simply prefer the look and features available with Prism, adding it to your theme is fairly straightforward.

  1. Download Prism with your desired options.

  2. Move the css and js files to your theme folder. I put prism.css in ./assets/css and prism.js in ./assets/js from the root.

  3. Update your functions.php file to "add_action" the files.

This is the code you need to add to functions.php to connect Prism to your theme:

function add_prism() {
    wp_enqueue_style( 'prism-css',  get_stylesheet_directory_uri() . '{path-to-prism.css}/prism.css' );
    wp_enqueue_script( 'prism-js',  get_stylesheet_directory_uri() . '{path-to-prism.js}/prism.js', [], false, true );
}
add_action( 'wp_enqueue_scripts', 'add_prism' );

The process of calling "add_action" and using "wp_enqueue_scripts" is the same method someone would use to add somehting like Font Awesome Icons, Google Fonts, or Bootstrap CSS to their theme.

Now you are ready to use Prism in WordPress, just use an HTML block with the "pre" and "code" tags and language class as described above.
